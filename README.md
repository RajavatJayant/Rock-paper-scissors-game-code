# ✊✋✌️ Rock Paper Scissors — Python CLI Game

A simple yet fun terminal-based **Rock Paper Scissors** game written in Python.
Play against the computer or go head-to-head with a friend!

---

## 🚀 Features

- 🤖 **vs Computer** — Random AI opponent
- 👥 **vs Friend** — Local two-player mode
- ✅ Input validation with helpful error messages
- ⚡ Lightweight — no dependencies, pure Python

---

## 🛠️ Requirements

- Python 3.x

---

## ▶️ How to Run

```bash
python rps.py
```
